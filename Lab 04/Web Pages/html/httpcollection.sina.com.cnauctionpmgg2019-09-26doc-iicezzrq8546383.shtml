9月28日至29日西泠拍卖杭州总部全门类征集藏品

2019年09月26日 13:11 新浪收藏
微博 微信 空间 分享 添加喜爱

　　本周六、周日（9月28日至29日），西泠拍卖杭州总部（杭州市上城区清吟街127号，平海路过中河高架向东150米）将举行全门类藏品公开征集活动，市民可携家中珍藏前往交流。秉承百年西泠印社文脉，西泠拍卖十五周年邀您共享艺术生活之美！

　　征集门类涵盖：中国书画、中外名人信札手稿、古籍善本、金石碑帖、名家篆刻及印石、珍品田黄、历代名砚、古董珍玩、文房瓷杂、青铜器（有来源证明）、明清家具、紫砂及茶道具、造像艺术、中国历代钱币、油画雕塑、摄影及电影艺术、名家漫画插图连环画、古代玉器、当代名家玉雕、和田玉籽料原石、西洋古董、珠宝首饰、中外陈年名酒。

　　本次征集为期两天，每日9：30至17：30，征集现场顺序安排，无需预约，详询：0571-87896778、87812580。

　　西泠印社拍卖有限公司杭州总部9月28至29日征集期间同时诚聘英才，应聘者可随带简历及证书复印件直接前往，职位介绍及联系方式参照“西泠拍卖官网常年招聘网页”。

　　西泠秋拍、西泠（绍兴）拍卖、西泠网拍同步公开征集。公司杭州总部及北京办事处、上海办事处常年征集。更多信息请关注西泠拍卖官网、微信公众号“西泠拍卖”、新浪微博“西泠印社拍卖”。

　　西泠拍卖再获“青花奖”五项大奖，四度蝉联年度十佳！

　　西泠拍卖荣获第四届“青花奖”五项殊荣

　　2019年9月24日，中国文物艺术品拍卖最高奖——第四届中国文物艺术品拍卖排行榜（青花奖）颁奖典礼在上海举行。西泠拍卖凭籍优秀业绩再度荣获“年度规模奖”、“年度创税奖”、“年度结算率奖”、“年度盈利率奖”，并连续四次蝉联“中国文物艺术品拍卖年度十佳企业”。西泠印社拍卖有限公司总经理陆镜清先生出席颁奖典礼并代表获奖企业发言。

　　当晚，中国拍卖行业协会（CAA）发布《2018中国文物艺术品拍卖市场统计年报》。年报权威数据显示，西泠拍卖在2018年度全国文物拍卖企业实收拍品款、实收佣金、主营业务利润、企业总纳税、产业链贡献、慈善捐款六项重要指标排名均位列行业前三。

　　“青花奖”设立于2016 年，由中国拍卖行业协会主办，主要依据每年度全国文物艺术品拍卖企业经营数据指标进行评选（行业合作奖除外），旨在树立我国文物艺术品拍卖市场健康导向，促进文物艺术品拍卖企业有质发展，提升我国文物艺术品拍卖行业主流形象。

　　文化寻根、自信传承：2019西泠春拍10.266亿热力收官！

　　2019年7月8日，西泠印社二〇一九年春季拍卖会以总成交额10.266亿元，总成交率86.26%圆满收官！得益于新老藏家的鼎力支持，春拍十大专场收获了九成以上的高成交率。

　　西泠拍卖成立15周年之际，秉承百年西泠印社“保存金石、研究印学，兼及书画”之创社宗旨，2019西泠春拍，融诗书画印于一体，广及诸艺术门类，立足民族优秀文化，传承共享、相得益彰，展现出百年印社在新时代的文化影响力。

　　·中国书画·

　　中国书画七大专场总成交额超过四亿元，平均成交率近九成，四件拍品过千万，稳健扎实，彰显了西泠拍卖在书画板块一如既往的市场购买力和对艺术品价值的挖掘力。

　　尤其是古代书画部分，亮点频出。古代书画两个专场，总成交额近两亿元，“石渠宝笈”著录赵孟頫《滚尘马图》以1265万元成交；“元四家”之一倪瓒《疏篁古木图卷》以1058万元成交；张正学旧藏、书学大家伊秉绶《隶书临蔡邕十字碑》60万元起拍，上至356.5万元成交；中国书画泥金扇片专场53件拍品，总成交额2400万元。近现代书画部分，海派大师谢稚柳巨幅杰作《为人民大会堂上海厅作虬松图》以1840万元成交，摘得全场桂冠；齐白石《一帆风顺图》以1104万元成交。

　　Lot 315 倪瓒疏篁古木图卷 成交价1058万

　　Lot 316赵孟頫 滚尘马图 成交价1265万

　　Lot 778 谢稚柳为人民大会堂上海厅作虬松图 成交价1840万

　　·名人手迹·

　　经过此前长期策划、宣传以及学术展览研讨、名家朗读等系列文化活动，各界瞩目的郁达夫唯一存世完整著作手稿《她是一个弱女子》，让世人重新记起了那位为中华民族抗战到底、为亚洲反侵略斗争献出生命的文学战士——郁达夫。此一著名中篇小说《她是一个弱女子》完整创作稿于2019年7月7日下午两时许在众人期待中开拍，经过多轮竞价，最终以897万元成交，创下郁达夫手稿拍卖最高纪录。

　　Lot 2403 郁达夫唯一存世完整著作手稿名作中篇小说《她是一个弱女子》完整创作稿 成交价897万

　　·古籍善本·金石碑帖·

　　陈公孟、郭若愚递藏《小字麻姑仙坛记》宋明拓本四种合册，内有宋拓南城原石本、南城本碑阴（七家小楷）、宋拓越州石氏本、明拓停云馆本、明拓玉兰堂本四种合璧，复经明代李日华、万寿国等九家题跋，著录于《善本碑帖录》等权威碑帖著作，保存完好，难能可贵。本拍品为识者所宝以345万元成交，创本季春拍碑帖门类最高成交价。吴稼农旧藏汤贻汾晚年重要诗词稿《琴隐集》、《师堀草》（部分未刊）则以103.5万元成交。

　　唐吐蕃统治时期敦煌写经《净名经集解关中疏（佛国品第一、方便品第二、弟子品第三）》为我国历代大藏经所未收，是研究中国佛教思想的一部重要参考著作，80万起拍，172.5万成交。金石考据大家张廷济旧藏颂敦等金石文字集萃2幅8万起拍，78.2万成交。

　　Lot 4758 郭若愚旧藏，李日华、万寿国等九家题《小字麻姑仙坛记宋明拓本四种合册》 成交价345万

　　本次西泠春拍“古籍善本·金石碑帖专场”，特设清代藏书家（LOT 4691-4716），推出严可均自校的经学代表作《唐石经校文十卷》，陶北溟题跋张穆手校本《元遗山先生集四十卷》，朱孔阳旧藏袁枚等八家为何春巢《卖花图》所作诗札，梁鼎芬、褚德彝等十家为缪荃孙题《扪碑读画图》，以及顾嗣立、何焯、沈德潜、法式善、阮元、齐召南、袁廷檮、丁丙、孔广陶、陆心源、周星诒、梁鼎芬、潘祖荫等信札、诗稿近二十种，以饷藏家。一经推出，便获得了藏家们的纷纷加价，其中袁枚等八家题何春巢《卖花图诗词札》（朱孔阳旧藏）以3万元起拍，藏家直接报价10万元后几番竞价，最终以43.7万元易主。

　　·名家篆刻·历代名砚·

　　2007年西泠拍卖开创性的设立名砚拍卖专场，以砚之形式，展现文人雅趣，获得业内关注。自此砚台门类成为一个收藏专项走入藏家视野，逐渐得到藏家的珍视，之后更是高价频现，引燃市场收藏热情。

　　2019西泠春拍作为西泠印社拍卖有限公司成立15周年的开篇，秉承百年西泠印社文脉，印砚部分依旧精彩纷呈，佳作涌现。此次共推出“文房清玩·历代名砚暨古墨专场”、“寄兴金石·名家闲章集萃专场”、“文房清玩·近现代名家篆刻专场”三大专场400余件精品力作，获得了现场藏家的激烈竞夺。其中《沈氏砚林》著录的吴昌硕铭，沈石友藏墨池砚以200万起拍，经过现场数十轮竞价最终以448.5万元成交，再显名家名砚之珍。清·赵国麟、林在峨铭董沧门作赤壁图随形端砚，雕刻精炼、铭文浩博，最终以126.5元成交。名家闲章集萃专场和近现代名家篆刻专场推出数方黄士陵刻章，其中清·黄士陵刻寿山石博古钮对章、清·黄士陵刻昌化鸡血石闲章均以112.7万元成交，赵叔孺刻青田石裘荫千自用印也以112.7万元的价格成交，创其篆刻作品拍卖纪录。多次出版的齐白石刻青田石王泽宽自用对章拍前就已备受关注，两枚体型硕大的齐白石刻印最终以103.5万元成交。

　　Lot 3820吴昌硕铭，沈石友藏墨池砚 成交价448.5万

　　Lot4009 赵叔孺刻青田石裘荫千自用印 112.7万创纪录成交

　　·文房瓷杂·著录青铜器·

　　2019西泠春拍文玩专场延续文人风雅，以物寄情，澄怀味象。“萃古熙今·文房古玩”、“文房清玩·古玩杂件”两大专场高溢价拍品频出，多件拍品以近估价十倍金额成交。其中萃古熙今·文房古玩专场甄选佳构深受藏家追捧，专场成交率高达91.95%。

　　Lot 891 清乾隆·翡翠雕花果鸣虫兽耳衔环瓶 80万起拍，552万成交。

　　西泠拍场之历代珍瓷门类较之国内瓷器拍场凸显自身特色。高古宋瓷定窑印花牡丹纹盘无论从器型、品相、纹饰来看，堪称完美无二，最终以264.5万成交，其显赫“背景”来源于日本顶级藏家、拍场上“金字招牌”临宇山人旧藏。

　　Lot 1889 宋·定窑印花牡丹纹盘 成交价264.5万

　　金石永年·重要青铜器及吴大澂御贡古驵琮专场两件重要作品持续引发关注，并取得拍场成交佳绩。西周初年古郯国遗珍颖簋，被经手无数瑰宝的端方盛赞颖簋“海内古今第一”、亲绘赠颖簋全形绘，且在《陶斋吉金录》首卷即收录之，自清末以来出版、著录近二十次，传承有绪，667万成交足见其珍。

　　lot811 西周早期·青铜颖簋 及端方原题跋颖簋全形绘立轴 260万起拍，667万成交

　　另一件目前市场已知愙斋珍藏玉琮之冠，不仅配以原匣原装的珍贵楠木锦盒，还在西泠印社2013年春拍第3136号王文心旧藏《吴大澂拓注金石各器屏》一组朱拓四条屏中有一条著录“组琮”拓片与此件形制一致，此件珍藏897万成交，曾经吴大澂向清内廷进呈，仅此一件不可再得。

　　lot812 御贡——吴大澂藏古驵琮 成交价897万

　　·紫砂及茶文化·

　　紫砂专场作为西泠拍卖传统的“人气专场”，拍卖当天买家热情高涨，座无虚席，重要拍品悉数高价成交，高溢价拍品频现，成交率达到91%。其中，一件清道光·瞿子冶刻吉安制月壶款紫泥调砂石瓢壶以391万元成交，此壶气韵高古，镌刻文笔潇洒自在，刀势生动，凌而不乱，是为子冶兴之所至之作。此外，玉成窑传世佳作竞价激烈，清晚期·玉成窑阿曼陀室款紫泥汉君壶超估价三倍成交。

　　Lot 3696 清道光·瞿子冶刻吉安制月壶款紫泥调砂石瓢壶 成交价391万

　　·古玉·当代玉雕·

　　玉器部分表现稳健，专场中一组采购自知名古董商和拍卖行的清·玉质鼻烟壶一组四十件，用料考究，纹饰精美，造型典雅，对研究鼻烟壶的发展历史、工艺制作、艺术风格、古时收藏审美等方面具有重要的参考价值。此组鼻烟壶以350万元起拍，以621万元的高价成交。当代玉雕大师作品专场中，中国玉石雕刻大师瞿利军的作品《携琴访友》白玉插牌也以201.25万元的高价觅得“知音”，玉雕艺术发展至今，其文化内涵已随时间推移而逐渐加深，方寸玉石之上也可传递高山流水的风雅之情。

　　Lot 1620 清·一组四十件 成交价621万

　　·珠宝翡翠·

　　珠宝部分，翡翠、钻石等传统强势版块表现不俗，钻石中一件“倾城”8.95克拉淡彩棕粉钻（FancyLight Brownish Pink）戒指也以575万元高价成交，翡翠版块中“风华绝代”冰种满绿手镯以600万元的价格起拍，经过多轮竞价，最终成交价突破千万。西泠拍卖珠宝部深入挖掘中国传统珠宝类拍品的文化价值与艺术价值，诸如象征东方气韵的翡翠、书写了历史的各类传统配饰，都是西泠较为推崇的特殊版块，且市场反应度上佳。

　　Lot 3369 “风华绝代”冰种满绿手镯 成交价1138.5万

　　此次西泠春拍其他各个场次亦不乏亮点，现当代油画雕塑专场、名家漫画·插图连环画专场、佛教艺术专场、历代钱币专场、陈年茅台酒专场拍品皆以高成交率悉数成交。其中，现当代油画雕塑专场成交率高达93.46%，吴大羽《春风拂清香》以667万成交。中国名家漫画·插画连环画专场中，雷德祖所作《斯巴达克思》连环画原稿以184万元的高价成交。历代佛教艺术专场备受关注的封面拍品“铜九顶佛坐像”是目前市场已知唯一一尊清代“六品佛楼”主佛，此件拍品以350万元起拍，经过场内外藏家激烈竞价，最终以920万元成交。优异的拍场表现再显藏家对西泠拍卖的认可和推崇。
